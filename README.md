## Estrutura de pastas do DEK

```
dekProject
│-- README.md
│-- index.js  
│-- main.js
|-- package.json
|-- .env.example
└───loaders
    │-- AfterLoad.js
    │-- BeforeLoad.js
```

## Instalação

```bash
npm i -g dek
```

### Criação de um novo projeto dek

```bash
dek init projetoDek
```


## Dando start no seu projeto

Após criar sua 

```bash
cd projetoDek
npm install
```
